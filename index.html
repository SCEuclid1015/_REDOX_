Overview

This project focuses on building a humanoid robot with capabilities like vision, movement, and object interaction. It uses the ESP32-CAM module for real-time video streaming and object recognition. High-torque servos control the robot's limbs for smooth, human-like movements, while a servo driver ensures coordination. A lithium battery powers the robot, and jumper wires connect various components, enabling enhanced interaction through AI.
